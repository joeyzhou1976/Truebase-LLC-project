## 文件夹说明

- `unsplash/`：从 Figma Make 源代码中解析出的页面照片素材。
- `named/`：已按用途整理命名的素材，包括 logo、预览图和 UI 图形。
- `raw/`：从当前 Figma 页面网络资源中原样抓取的文件。
- `manifest/`：资源 URL、代码快照、二进制文件 manifest 等记录。
